**项目名称**：中国高等教育机会的经济效应分析  
负责人:尚金朝 10245501440    
**立项原因**：  
    我们宿舍三个山河四省的学生（山东山西河南河北）,大家都知道自己省份卷得颇具盛名;尤其是河北孩子,据说由于诸多原因河北境内没有一所985本校,大家只能一边和衡水竞争一边往外考,还穷,于是我产生了对教育资源不均匀分配的朴素认识,并由此产生了对教育和经济情况综合考虑和数据分析的想法.  
    另:2024年当年,北京市的总GDP超过了河北省的总GDP,二者毗邻但经济教育情况大不相同,其也是这个作业的启发之一.  
**数据来源**:  
    本报告绝大部分数据，如各省的GDP，教育支出，人口来自国家统计局(https://data.stats.gov.cn ),另外的录取率等则来自各省教考院，全部原始数据放在文件**DaSE大作业素材**的**OData.xls**中   
**数据处理**：  
    数据处理是用excel完成的，其也是统计局提供的下载格式，全部数据处理及结果在**DPsheets.xlsx**中  
    在处理文件中采取了分层的策略如下：  

      
 数据->基础指标->分析计算->结果   
  
数据层  
洗出有用数据：2024GDP_亿元，	 2024人口_万人，	 2024教育经费_万元， 	2024出生率_‰	，  985录取率，	 211录取率，	  双一流录取率  
指标层
计算基础指标：人均GDP_元，	教育投入强度_%，	人均教育经费_元，	人口压力指数，	效率比值，	公平系数，	985排名
计算层   
包含三个sheet，分别计算了：最值排名相关系数，教育投入强度与985录取率的效率关系，公平性的洛伦兹曲线等   
其在excel自带的数学运算上引入了高级统计公式如基尼系数(经济系数活用)，因为数据性质各种作比求和处理如**效率比值 = 985录取率 / 教育投入强度**等繁杂难以例举   
结果层   
就是总结，使excel完整程度提升

**项目局限**：
   不可避免的这种庞大题材有一个报告写不下的局限性，造成地方经济和教育差距的原因有很多：比如各地区能录取多少985其实很大程度上是名额决定的，当然名额是一个很表面的数据，但是名额又很大程度上受大学所在地影响，这就导致有院校—人口比上有绝对多985等优秀学校的北京不可避免地会有冠绝的录取率。当然还有政策青睐等诸多因素不一而足。   
   虽然有很多在考察时就体现的问题，但该项目的可视化价值仍然不小，直观的数据对比除了让人感叹一句“是条狗都得生在北京”之外，大概还能多少揭示在优质教育资源不平均的水面下，内卷着挣扎着的芸芸学子们的参差
